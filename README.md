## 🔎기능

- H&M 로고를 클릭하면 메인 페이지로 이동합니다.
- 로그인 버튼을 클릭하면 로그인 페이지로 이동합니다.
- 검색창에 티셔츠,드레스 등 키워드를 입력하면 일치하는 상품이 검색됩니다.
- 상품 상세페이지를 보려면 로그인을 해야합니다.
- 로그인 상태에 따라 로그아웃/로그인 버튼이 다르게 보입니다.
